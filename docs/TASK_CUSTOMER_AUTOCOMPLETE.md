# Task: Customer Autocomplete

## Objective
Standardize customer selection using searchable dropdown across all relevant screens.

## Requirements
1. Replace Customer dropdown with searchable autocomplete
2. Support search by:
- CustomerCode
- CustomerName
- TaxId
3. Show customer information after selection
4. Use same UI pattern across Quotation, Invoice, Payment, and Receipt related pages

## Constraints
- Do not change business logic
- UI/UX only
